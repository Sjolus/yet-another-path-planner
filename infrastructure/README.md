# Infrastructure

This directory contains Kubernetes manifests, Helm charts, and Terraform configurations for deploying Yet Another Path Planner.

## Structure

- `helm/` - Helm charts for Kubernetes deployments
- `kustomize/` - Kustomize overlays for environment-specific configs
- `terraform/` - Terraform modules for infrastructure provisioning

## Getting Started

Helm charts and Kubernetes manifests will be added as services are developed and ready for deployment.

See `docs/architecture.md` for the complete deployment strategy.
