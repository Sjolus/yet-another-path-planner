== Important Notes ==

  1. NEXT_PUBLIC_* environment variables are baked into the frontend Docker image
     at build time. Changing them via ConfigMap only affects server-side rendering.
     To change client-side values, rebuild the frontend image.

  2. For production, replace the Kubernetes Secret with External Secrets Operator
     or SealedSecrets resources pointing to your secrets manager.

  3. Default secrets (JWT, database password) are NOT production-safe.
     Always override via --set or a separate values file.
